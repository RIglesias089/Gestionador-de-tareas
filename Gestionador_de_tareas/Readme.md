En este proyecto buscaremos crear y documentar la creacion de el programa gestionador de tareas
y de recordatorios colaborativos para distintos tipos de usuarios registrados, permitiendo realizar
acciones segun el tipo de usuario usando POO, patrones de disenio, concurrencaias interfaces graficas
con JavaFX y conexiones a bases de datos para ofrecer una solucion escalable.

Adjuntaremos el desarrollo de los diagrams respectivos para el correcto analicis de el proyecto.
